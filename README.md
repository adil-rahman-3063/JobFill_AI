# 🚀 JobFill AI - The Premium AI Job Application Assistant

**JobFill AI** is a high-performance Chrome extension that leverages the **Gemini API** (v1beta) to automate and simplify job applications. Designed with a premium aesthetic and a "Privacy First" mindset, it helps you apply to jobs faster by identifying complex questions and learning from every form you fill.

![JobFill AI Logo](assets/icon128.png)

## ✨ New & Advanced Features

- **🎓 Learning Mode (Remember This?)**: Detected a new question? JobFill AI can "learn" and remember your answers for future applications with a simple toggle.
- **🛡️ Selective Autofill**: Respects your privacy and time by automatically filling **only mandatory (*) fields**, leaving optional ones for you.
- **🌍 Dynamic Model Discovery**: Automatically identifies the best available Gemini model for your API key (supporting Gemini 1.5, 2.0, 2.5, and 3.0).
- **📋 Comprehensive Eligibility Mapping**: Built-in support for complex legal questions:
  - **Work Authorization**: Country-specific authorization matching.
  - **Visa Sponsorship**: Intelligent handling of sponsorship requirements.
  - **Veteran & Disability Status**: Accurate mapping for diversity reporting.
  - **Salary & Relocation**: Pre-configured preferences for quick submission.
- **💎 Premium Glassmorphic Design**: A stunning, docked sidebar that pushes page content so you can see the application form clearly while analyzing job insights.
- **📂 Secure Resume Management**: Locally stored resume data for intelligent, private processing.

## 🛠️ Tech Stack

- **Core**: JavaScript (Manifest V3)
- **AI**: Google Gemini API (v1beta support for latest Flash/Pro models)
- **Styling**: Vanilla CSS with Modern Glassmorphism & Micro-animations
- **Storage**: Chrome Storage API (100% Local)

## 🚀 Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/adil-rahman-3063/JobFill_AI.git
   ```
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (toggle in the top right).
4. Click **Load unpacked** and select the `jobfill_ai` folder.
5. Click the extension icon and select **Configure Profile** to start.

## 🔑 Configuration

To use the AI features, you'll need a **Gemini API Key**.
1. Get your free key from [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Paste it into the **AI Configuration** section of the JobFill AI setup page.
