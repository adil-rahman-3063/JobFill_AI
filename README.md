# 🚀 JobFill AI - The Premium AI Job Application Assistant

**JobFill AI** is a high-performance Chrome extension that leverages the **Gemini 1.5 Flash API** to automate job applications. Designed with a premium aesthetic and a "Privacy First" mindset, it helps you apply to jobs faster by analyzing descriptions and autofilling forms with one click.

![JobFill AI Logo](assets/icon128.png)

## ✨ Features

- **🧠 Intelligent Job Analysis**: Automatically extracts Company Name, Role, and Required Skills from any job page.
- **⚡ One-Click Autofill**: Maps your personal profile and resume data to form fields across major job platforms (LinkedIn, Indeed, Lever, Greenhouse, etc.).
- **💎 Premium Design**: A stunning, glassmorphic sidebar UI that integrates seamlessly with your browsing experience.
- **🛡️ Privacy First**: Your sensitive data (API keys, resume, etc.) is stored locally in your browser. No external servers involved.
- **🚀 Onboarding Experience**: A smooth, multi-step setup flow to get you started in seconds.

## 🛠️ Tech Stack

- **Core**: JavaScript (Manifest V3)
- **AI**: Google Gemini 1.5 Flash API
- **Styling**: Vanilla CSS with Modern Design Tokens
- **Storage**: Chrome Storage API

## 🚀 Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/jobfill-ai.git
   ```
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (toggle in the top right).
4. Click **Load unpacked** and select the `jobfill_ai` folder.
5. Follow the **Onboarding Flow** that appears automatically.

## 🔑 Configuration

To use the AI features, you'll need a **Gemini API Key**.
1. Get your free key from [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Paste it into the **AI Configuration** section of the JobFill AI setup page.
